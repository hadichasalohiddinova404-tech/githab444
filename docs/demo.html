Husanboy NFT dApp

NFT minting, metadata boshqaruvi va community voting oqimini bitta Web3 mahsulotga birlashtiruvchi demo loyiha.

Muammo va yechim

  • Oddiy NFT loyihalarda mint va governance ko‘pincha alohida platformalarda yuradi.
  • Bu dApp ikkisini birlashtirib, creator community uchun yagona tajriba beradi.

Smart-kontrakt

  • ERC-721 NFT minting
  • Mint fee
  • Token URI update
  • Proposal create, vote, execute

Frontend demo oqimi

  • MetaMask wallet ulash
  • NFT mint qilish
  • Proposal yaratish
  • Yes/No vote berish
  • Deadline’dan keyin execute qilish

Natija

Hardhat testlar o‘tgan, React frontend build bo‘lgan, Git repo tayyor. Real Sepolia deploy va GitHub push uchun RPC, private key va remote kerak bo‘ladi.
